Privacy Policy

Last updated: December 2025

1. Overview

This Privacy Policy explains how Furina collects, uses, and protects data when used in Discord servers. The bot operates only within servers where it is explicitly invited.

2. Data Collected

  • User, Server, and Channel IDs for core functionality
  • Message content for moderation commands, AFK systems, counting channels, and deleted message sniping
  • Member data such as roles and nicknames
  • Optional presence data for opt-in status-based role features

3. How Data Is Used

  • Executing commands and interactions
  • Moderation and server management
  • Utility tools, games, and giveaways
  • Optional AI-powered responses when explicitly requested by users

Data is not used for advertising, analytics, profiling, or AI training.

4. Data Storage and Retention

  • Message content may be temporarily stored in a local JSON file only for moderation-related features such as message sniping. Stored data is automatically deleted after a maximum of 30 days.
  • AI command memory is stored only in short-term runtime memory and is cleared automatically.
  • Configuration data may be stored to support persistent settings.

5. Data Sharing

No data is sold or shared with third parties. AI commands send user-provided input to a third-party AI service (Google Gemini) only when explicitly invoked.

6. User Control

Server administrators can configure or disable features at any time. Users may request deletion of stored moderation data by contacting the developer.

7. Security

Reasonable technical measures are taken to protect processed data. The bot follows Discord’s Developer Terms of Service and API policies.

8. Changes

This policy may be updated as features evolve. Continued use of the bot indicates acceptance of updates.

9. Contact

For questions or concerns, contact @twin.67 on Discord.
